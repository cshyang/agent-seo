# Hermes Adapter

Hermes can use this repo as a skill pack and methodology source.

## Install options

- Copy selected `skills/*` directories into the active Hermes profile.
- Keep this repo checked out and load its workflow docs during report generation.
- Use a future sync script to install/update skills from this repo.

## Notes

- Skills in this repo are reusable workflow definitions, not client memory.
- Do not commit profile-local credentials or generated report outputs.
