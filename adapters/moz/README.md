# Moz Adapter

Status: optional enrichment.

Moz API can provide lightweight authority and link metrics. Use it for context, not as a default monthly KPI.

## Intended uses

- Onboarding authority sanity check.
- Competitor DA/PA and linking root domain context.
- Spam Score review when a domain has suspicious link history.
- Backlink/source hints when no stronger link dataset is available.

## Free/low-tier caution

Moz API limits can be tight. Treat calls as scarce and record when they are used.

## Reporting rules

- Do not lead client reports with DA/PA.
- Do not report tiny monthly DA/PA changes as progress.
- Do not recommend link building from Moz metrics alone.
- Convert findings into concrete opportunities, e.g. missing directory/source presence, suspicious spam pattern needing review, or competitor source gap.
