# Quarterly Refresh Workflow

Quarterly checks prevent monthly reports from becoming bloated while still catching structural changes.

## Quarterly jobs

- Competitor keyword gap via DataForSEO Labs.
- LLM Mentions Top Domains/Top Pages.
- On-Page technical health crawl.
- Google Business Profile/review benchmark where local intent is high or medium.

## Promotion rule

If a quarterly check finds a P0 issue such as deindexing, site-wide noindex, Core Web Vitals collapse, or broken enquiry tracking, it overrides the normal monthly diagnosis.

## Output

- Appendix-only reference data unless it changes the monthly diagnosis.
- Section 6 action only when the check produces a concrete, client-executable next step.
