# External SEO Enrichment

External enrichment sources can support diagnosis and discovery, but they do not become default monthly KPIs unless they produce a concrete action.

## Core vs enrichment

Core monthly sources:

- GSC: query clicks/impressions and query-page mapping.
- GA4: organic sessions and enquiry events.
- DataForSEO: stable rank/SERP/local/GEO observations.
- Client feedback: lead quality truth.

Optional enrichment sources:

- Common Crawl web graph / index data.
- Moz API link metrics.
- Bing Webmaster, Ahrefs, Semrush, directory checks, and other third-party data when available.

## Common Crawl use cases

Use for:

- Source discovery: third-party pages, directories, and competitor mentions.
- Historical web presence context.
- Domain-level backlink/source graph hints.
- GEO research: pages and domains that may shape AI answers.

Avoid for:

- Monthly KPI reporting.
- Fresh ranking or visibility conclusions.
- Exact backlink counts.
- Client-facing health scores.

## Moz API use cases

Use for:

- One-time authority sanity checks during onboarding.
- Competitor context: DA/PA, spam score, linking root domains.
- Lightweight backlink/source gap hints.

Avoid for:

- Monthly DA/PA movement commentary.
- Link-building recommendations without concrete source opportunities.
- Replacing GSC/GA4/DataForSEO.

## Reporting rule

Only surface enrichment when it changes the recommendation. If enrichment finds nothing actionable, keep it in the appendix or omit it.
