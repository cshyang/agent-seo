# Methodology

Agent SEO reports should behave like an operating loop, not a dashboard dump.

## Reporting doctrine

1. Enquiries and commercial demand come first.
2. Separate branded demand from non-brand commercial SEO performance.
3. Use fixed money keyword and AI/GEO prompt panels for comparability.
4. Use DataForSEO for stable rank/SERP observations; use GSC for actual query clicks/impressions; use GA4 for enquiry/session truth.
5. Make one diagnosis per month.
6. Recommend two to three actions only.
7. Record blockers and grade last month’s actions.
8. Treat low-volume movement cautiously and label data confidence.

## Data-source responsibilities

- GSC: query/page mapping, impressions, clicks, gap discovery.
- GA4: organic sessions, key events, landing page outcomes.
- DataForSEO SERP: fixed-location/device ranks, ranking URL, SERP blockers, local pack presence.
- DataForSEO Labs: volume/difficulty validation and quarterly competitor gaps.
- DataForSEO AI Optimization: fixed prompt panel, mentions/citations, top sources.
- Client/CRM: lead quality validation.

## Deliberately excluded by default

- Monthly backlink reports.
- Toxic link reports except one-time onboarding checks.
- AI keyword volume as a headline metric.
- Google Trends unless seasonality is not known.
- Large technical audits every month.
