# DataForSEO Cost Control

Use DataForSEO deliberately. The goal is stable commercial diagnosis, not maximal API coverage.

## Monthly default

Reuse one SERP response per money keyword for:

- true rank
- ranking URL
- target URL mismatch/cannibalization
- local pack presence
- AIO/local pack/ads blockers
- visible SERP competitor observations

Use Labs only for Section 7 opportunity validation unless a deeper diagnostic is explicitly needed.

## Quarterly default

Run quarterly only unless requested:

- competitor keyword gap
- LLM Mentions Top Domains/Top Pages
- On-Page crawl
- Google Business Profile/review benchmark

## Report logging

Every report should record:

- source jobs run
- task count or estimated API cost
- skipped jobs
- missing data sources
- data confidence caveats

## Avoid by default

- Backlinks API
- Content Generation API
- AI Keyword Data / AI search volume
- repeated full-domain crawls
